## [1.0.0] - 2025-12-02
### Added
- First stable release of **PayBuddy**
- `PayBuddy::processPayment()` method for handling basic payment logging
- PSR-4 autoloading support via Composer
- Yii2 integration with `Yii::info()` logging
- Example configuration file for registering PayBuddy as a Yii component
- Basic PHPUnit test suite
